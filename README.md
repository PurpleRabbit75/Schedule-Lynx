## Quick Summary

Schedule Lynx is a cross-platform project designed to help groups of ~ a dozen people compare their weekly schedules. It is especially helpful for college students, but can be used by anyone anywhere.

## Installation


To install Schedule Lynx, download it in any way you wish. You can click the green <> Code button above and download it as a zip, or git clone if you're more comfortable with that. If you download it as a zip, unzip it. 

Congrats! Schedule Lynx is now installed!

## Inputs and Outputs

Schedule Lynx takes in a pile of .json files and spits out a .jpg file. You specify which files come from and go to which directories by editing config.json.

## Standard Operating Proceedure

To create a new schedule, first create a new folder. In this folder, place person.json file. You can create a person.json file by copying the person_template.json file and editing it to match the schedule of any given person.
